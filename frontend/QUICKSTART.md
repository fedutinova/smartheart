# Quick Start Guide

## Быстрый старт фронтенда SmartHeart

### 1. Установка зависимостей

```bash
cd frontend
npm install
```

### 2. Настройка окружения

Создайте файл `.env` (необязательно, если бэкенд на localhost:8080):

```bash
VITE_API_URL=http://localhost:8080
```

### 3. Запуск приложения

```bash
npm run dev
```

Приложение будет доступно на http://localhost:3000

**Важно:** Убедитесь, что бэкенд запущен и CORS настроен правильно.
Если возникают CORS ошибки, перезапустите бэкенд после последних изменений:
```bash
# В корне проекта
CGO_ENABLED=1 go run cmd/main.go
```

### 4. Первые шаги

1. Зарегистрируйте нового пользователя на странице `/register`
2. Войдите в систему на странице `/login`
3. Откройте Dashboard - главную страницу
4. Перейдите в "Анализ" для загрузки EKG изображения
5. Просмотрите результаты в "История"

### 5. Основные страницы

- `/login` - Вход в систему
- `/register` - Регистрация
- `/dashboard` - Главная страница с последними анализами
- `/analyze` - Форма для анализа EKG
- `/history` - История всех анализов
- `/results/:id` - Детали конкретного результата

### 6. Технологический стек

- **React 18** - UI framework
- **TypeScript** - Типизация
- **Vite** - Сборщик
- **React Router** - Маршрутизация  
- **React Query** - Серверное состояние
- **Zustand** - Глобальное состояние
- **Tailwind CSS** - Стили
- **Axios** - HTTP клиент

### 7. Сборка для production

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

### 8. Troubleshooting

**Проблема: не подключается к бэкенду**
- Проверьте, что бэкенд запущен на порту 8080
- Проверьте переменную окружения `VITE_API_URL`

**Проблема: ошибки при сборке**
- Удалите `node_modules` и установите заново: `rm -rf node_modules && npm install`
- Проверьте версию Node.js (рекомендуется 18+)

**Проблема: зависимости не устанавливаются**
- Если возникают конфликты, все уже исправлено в package.json
- Просто запустите `npm install` еще раз

**Проблема: стили не применяются**
- Проверьте, что Tailwind CSS установлен: `npm list tailwindcss`
- Пересоберите проект: `npm run build`
